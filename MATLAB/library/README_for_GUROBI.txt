<구로비 사용법>

1. go to 'http://www.gurobi.com/index'
2. make user ID (email, '@snu.ac.kr' is recommanded)
3. verify yourself as a student by snu email
4. go to menu 'download > license' and press student license button at the middle of the page
	- make your license
	- note that you cannot use the registered license at different machine
	- you can make multiple licenses
5. download the latest gurobi software
6. register your license with using the command (window key + R) window
